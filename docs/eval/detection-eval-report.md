# Detection evaluation

Generated by `python -m benchmarks` against a deterministic synthetic
corpus. The index is seeded with each scam campaign's base image; scam
variants are perturbed re-shares that should still match, and clean
images should not. An image is *flagged* when its best ensemble score
(lower = closer match) is at or below the threshold.

Corpus: 6 campaigns, 48 scam images (6 bases + 42 variants), 18 clean negatives.

## Recommended operating point

- **Score threshold:** 0.3000
- **Precision / Recall / F1:** 1.000 / 0.792 / 0.884
- **FPR:** 0.000  (FP=0, on 18 clean images)
- **Rationale:** highest recall (0.792) at zero false positives; score threshold 0.3000
